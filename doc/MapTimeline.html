Class MapTimeline

Object
   |
   +--MapTimeline
class MapTimeline


This MapTimeline class represents an object that gives a single entry point to synchronize a Simile Timeline and OpenLayersMap.
Defined in MapTimeline.js

void addEvent(<String> guid, <String> start, <String> end, <String> title, <String> description, <Decimal> lat, <Decimal> lon, <String> image, <String> link)
    Adds a data point to map and timeline
    Parameters:
      guid - Unique global identifier
      start - (optional) date in the format '2008-04-13 12:58Z' or null. If no start then it will only be displayed on the map.
      end - (optional) date in the format '2008-04-13 12:58Z' or null
      title - Text to display on timeline
      description - Text displayed in popup bubble
      lat - (optional) Latitude
      lon - (optional) Longitude
      image - (optional) URL of image to display in popup bubble
      link - (optional) URL of link to display in popup bubble

initTimeline

void initTimeline(<String> div, <Timeline.createBandInfo[]> bandInfos, <Timeline.HORIZONTAL> orientation, <Timeline.DefaultEventSource> eventSrc, <Integer> bandIdx)
    Initializes Timeline see http://simile.mit.edu/timeline/docs/ for more options
    Parameters:
      div - Id of an element in your page that will contain the map.
      bandInfos - is an array of Timeline.createBandInfo objects
      orientation - of the timeline (Timeline.HORIZONTAL | Timeline.VERTICAL)
      eventSrc - an event source that provides events to be painted on this band, e.g., new Timeline.DefaultEventSource(). It can be null, which means the band is empty.
      bandIdx - an index into bandInfos that we will associate event handlers with
