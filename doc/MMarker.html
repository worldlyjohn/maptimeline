class MMarker


Represents a marker to be displayed on an OpenLayers map. Also has an associated popup This class references the OpenLayers Map API and Simile Timeline API. For more information see OpenLayers and Simile Timeline
Defined in Markers.js

MMarker

MMarker(<String> id, <OpenLayers.LonLat> point_in, <String> html_in, <OpenLayers.Icon> icon_in, <OpenLayers.Layer> layer_in, <OpenLayers.Feature.Vector> featureVectors_in, <Function> callBack)
    Parameters:
      id - A unique string to identify this marker
      point_in - longitude/latitude location of the marker
      html_in - HTML text to be displayed in the associated popup
      icon_in - Icon for marker
      layer_in - Layer for the popup feature to be displayed on (typically a WMS layer)
      featureVectors_in - Array of feature vectors associated with the marker (typically geoRSS). May be null
      callBack - function to call when map marker is clicked
